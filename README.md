# TextDetective

A procedurally generated detective game where you solve criminal cases using an expanding investigative toolkit. Hidden meta-narratives — organised crime networks, serial offenders — span multiple cases and must be discovered by the player.

## Overview

- **Genre:** Detective / mystery, procedurally generated
- **Platform:** Web, desktop, mobile (menu-driven UI)
- **Player count:** Single-player (multiplayer co-op planned for future)

## Gameplay

Cases are presented with an initial set of facts. The player uses investigative tools — forensic analysis, interviews, surveillance, warrants — to expand the evidence and reach a conclusion. Efficiency matters: unnecessary actions are penalised. Over time, patterns across cases reveal larger criminal narratives that the player can choose to pursue.
